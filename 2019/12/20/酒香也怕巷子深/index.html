酒香也怕巷子深

Yicheng Zhao 发布于 2019-12-20

  古往今来，“酒香不怕巷子深”，似乎是一条真理。”王婆卖瓜，自卖自夸“，对增加瓜的销量没有太大的帮助，反倒传为笑谈；然而那些执著于品质的百年老店，即使不事张扬，照样有着良好的口碑。可是，随着时代的发展，这一切仿佛成为了过去时。君不见，多少品牌砸几个亿去电视台播广告，挤占你看电视剧的时间？我们正处于一个“酒香也怕巷子深”的时代。也许你是块会发光的金子，但若是别人看不见你的光，你的努力终究是徒劳。

  《娱乐至死》里说到，今天是一个浮躁的时代，人们没有兴趣去深入了解一个萍水相逢的事物。我深以为然。一些百年老店仍在延续过去的生存模式，靠高品质来维持好口碑，从而吸引顾客。这条路在今天已经不好走了。许多“网红店”依靠精心设计的营销策略，大量投放的广告文案一夜爆红，每天都有数十万的销量。不少百年老店，却逐渐无人问冿。“网红店”的产品真的都优于百年老店吗？非也。这只是因为，网红店投入了大量的资金在营销上，这恰恰能吸引这个浮躁时代的顾客，反观百年老店，虽坐拥着高品质和高口碑，却从不刻意讨好顾客。过去的顾客们可能会耐心地尝试他们的产品，而今天的顾客却没有那么多精力了。百年老店的招牌不能让他们驻足；如此，他们又怎么可能真正走进那破旧的店面，去发现其中产品的美呢？在一个人们总是浅尝辄止的时代，酒香也怕巷子深才是赤裸裸的的真理。

  我们在前面提到了广告。的确，广告在这个时代是一个至关重要的因素。顾客要判断一个商品的好坏，亲自尝试当然是最好的选择。然而，正如前面所说的，今天的顾客已浮躁到无睱完全关注商品的真实质量。那是什么，才能左右顾客的抉择呢？当然就是广告！在今天这个商业发达的年代，任何一个商人都不可能忽视广告的力量。只有在电视上打出你的广告，你的商品才可能被众人熟知。从前的百年老店，可能满足于一个封闭地域内的一个固定的顾客群体，而今天的老板们，却不得不走向更大的天地。广告，正能让更多素不相识的人们知道你的产品，这样他们才能成为你潜在的顾客。如果你有酒香，请不要局限于狭小的巷子之中，只有让更多人都知道你，你的生意才可能做大做强。

  其实，何止是经商之人，才会害怕深巷？在现代社会，一个有才能的人何尝不需要推销自己？也许你不说出来，没有人知道你有多厉害；毕竟，酒香也怕巷子深；毕竟，是金子也怕蒙尘。我们何不自信地展示出自己的力量呢？

  这是一个酒香也怕巷子深的时代，这是一个游戏规则已经改变的时代。“善动者，动于九天之上”，过去那种身居陋巷而不忧的想法，已不足为我们采纳。是金子总会发光，而我们要在对的地方发光。

赵奕诚执笔于2019年3月
